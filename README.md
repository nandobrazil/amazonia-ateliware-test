# AmazonIA Ateliware Web

AmazonIA Ateliware Web is a modern web application made in Angular 16 to calculate the shortest path between 3 points.

## Installation

Use node v18.16.0.  Navigate to http://localhost:4232/
```bash
npm install
```

```bash
npm start
```
